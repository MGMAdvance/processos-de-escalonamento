# Processos de Escalonamento

Desenvolver quarto algoritmos de escalonamento de processos e serem apresentado em sala de aula.

- FCFS;
- SJF (Não-preemptivo);
- Algorítmo de Prioridades;
- Round Robin.

Os algoritmos serão desenvolvidos em **Java**.
